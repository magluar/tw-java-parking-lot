[] Case2

When fetching the car

[] Case3  
Given a car and a wrong ticket or no ticket

Then return no car  

[] Case4  
Given a car and a used ticket  
When ticket is already used  
Then return no car  

[] Case5  
Given a car and a parking lot  
When parking lot is full  
Then return no ticket
